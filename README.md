# Clothing Store Management System

This is a simple Java console project created for the Object-Oriented Programming course.

##  Project Description
The Clothing Store Management System simulates a basic clothing store.  
It demonstrates core OOP concepts such as classes, objects, constructors, and methods.

##  Features
- Create clothing items (name, size, price)
- Apply discount to clothing items
- Create customers and add bonus points
- Create orders and calculate total price
- Display information using `toString()`

- Java
- IntelliJ IDEA
